## EGO_ARK

一款基于Godot引擎开发的移动端音乐节奏游戏

### ✨ 主要功能
- 基础音符判定系统
- HOLD音符逻辑（开发中）
- 自动演奏模式（开发中）
- 角色加载器框架
- 可配置的判定偏移调整
- 简易设置菜单（开发中）

### 🛠️ 开发环境
- Godot 4.x
- 移动端手势适配
- 模块化代码结构

### 📌 待实现功能
